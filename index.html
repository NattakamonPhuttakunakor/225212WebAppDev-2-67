หลักสูตร: หลักสูตรวิทยาศาสตรบัณฑิต สาขาวิชาวิทยาการคอมพิวเตอร์ 2565

รายละเอียดวิชา (ภาษาไทย) พื้นฐานการพัฒนาเว็บ การพัฒนาส่วนติดต่อผู้ใช้ การออกแบบเว็บให้แสดงผลทุกขนาดหน้าจอ จาวาสคริปต์ การพัฒนาส่วนจัดการเว็บไซต์ การติดต่อกับฐานข้อมูล การเพยแพร่เว็บ กรณีศึกษาการพัฒนาเว็บไซต์ * พื้นฐานการพัฒนาเว็บ: สอนความรู้เบื้องต้นเกี่ยวกับการสร้างเว็บไซต์ เช่น โครงสร้าง HTML, CSS * การพัฒนาส่วนติดต่อผู้ใช้ (Front-end): สอนการออกแบบและพัฒนาส่วนที่ผู้ใช้เห็นและโต้ตอบด้วย เช่น ปุ่ม, รูปภาพ, เมนู * การออกแบบเว็บให้แสดงผลทุกขนาดหน้าจอ: สอนเทคนิคการทำให้เว็บไซต์แสดงผลได้ดีบนอุปกรณ์ต่างๆ เช่น คอมพิวเตอร์, แท็บเล็ต, สมาร์ทโฟน * JavaScript: สอนภาษา JavaScript ซึ่งเป็นภาษาที่ใช้ในการสร้างเว็บไซต์ให้มีความเคลื่อนไหวและโต้ตอบได้มากขึ้น * การพัฒนาส่วนจัดการเว็บไซต์ (Back-end): สอนการสร้างส่วนหลังของเว็บไซต์ เช่น การจัดเก็บข้อมูล, การประมวลผลข้อมูล * การติดต่อกับฐานข้อมูล: สอนวิธีการเชื่อมต่อเว็บไซต์กับฐานข้อมูลเพื่อจัดเก็บข้อมูลต่างๆ * การเผยแพร่เว็บ: สอนวิธีการนำเว็บไซต์ที่สร้างเสร็จแล้วไปเผยแพร่บนอินเทอร์เน็ต * กรณีศึกษา: สอนผ่านตัวอย่างการพัฒนาเว็บไซต์จริง เพื่อให้เข้าใจหลักการและแนวคิดในการพัฒนาเว็บไซต์ได้ดียิ่งขึ้น
